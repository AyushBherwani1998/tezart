Tezart

A library for building decentralized applications in dart language, currently focused on the Tezos platform.

What it is

Tezart connects to interact with the Tezos blockchain.

It connects to a tezos node to send transactions, interact with smart contracts and much more !

See the Quick start guide for more details.
